# 0x06 — PHP and the Web

## Environment Setup
```bash
# Install PHP with built-in server support
php -S localhost:8000
# Run this in your task directory — then visit http://localhost:8000
```

## What You Will Learn
- PHP superglobals: $_GET, $_POST, $_SERVER, $_COOKIE, $_SESSION
- HTTP headers in PHP
- Building a router from scratch
- Handling JSON API requests
- Sessions and authentication
- CSRF protection
- Rate limiting
