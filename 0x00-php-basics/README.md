# 0x00 — PHP Basics

## Environment Setup
```bash
# Install PHP on Linux Mint/Ubuntu
sudo apt update
sudo apt install php8.2-cli php8.2-common

# Verify installation
php --version

# Run a PHP file
php filename.php
```

## What You Will Learn
- PHP syntax and how it differs from other languages
- Variables, data types, and type juggling
- Operators and expressions
- Conditionals and loops
- How PHP executes line by line
