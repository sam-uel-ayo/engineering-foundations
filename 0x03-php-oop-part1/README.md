# 0x03 — PHP OOP Part 1

## What You Will Learn
- Classes and objects
- Properties and methods
- Visibility: public, protected, private
- Constructors and destructors
- Static properties and methods
- Magic methods
- Inheritance and method overriding
- Abstract classes
- Interfaces

## Why This Matters
Laravel is built entirely on OOP. Every Model, Controller, Service,
Job, and Event is a class. You cannot understand Laravel without
understanding PHP OOP deeply.
