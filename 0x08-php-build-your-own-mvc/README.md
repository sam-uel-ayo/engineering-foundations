# 0x08 — Build Your Own MVC

## What You Will Learn
By building your own MVC framework, you will understand every
decision Laravel made. When you finally use Laravel, nothing will
feel like magic. You will know exactly what is happening.

## The Goal
A working mini framework with:
- Autoloading
- Routing
- Controllers
- Request/Response objects
- Middleware pipeline
- Service container (from 0x04)
- Eloquent-like ORM (from 0x03 + 0x07)
- View rendering
- Session handling
