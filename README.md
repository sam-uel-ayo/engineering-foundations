# Engineering Foundations

> "I have to lock in now so I don't build my empire on lies." — 2026

This repository is my deliberate path from conceptual understanding to genuine fluency.
Every task is written by hand. No AI writes the solutions. Docs are allowed. AI is not.

## Rules
1. No AI for solution code. Ever.
2. Docs are allowed — PHP.net, MDN, Laravel docs, Node docs, Stack Overflow.
3. Each completed task gets its own commit. Not batched.
4. If a task takes more than 2 hours, go back one task. The prerequisite is missing.
5. Hard is the point. Discomfort means growth.
